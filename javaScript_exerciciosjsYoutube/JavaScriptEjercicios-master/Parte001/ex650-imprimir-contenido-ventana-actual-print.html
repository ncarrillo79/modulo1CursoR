JavaScript

JavaScript es un lenguaje de programación basado en scripts. Permite crear páginas y aplicaciones Web dinámicas.

Este lenguaje cuenta con una amplio ecosistema de librerías y frameworks. Se le puede considerar uno de los lenguajes de programación más populares en la actualidad.

Con JavaScript un programador, puede además, crear aplicaciones móviles para las diferentes plataformas móviles: Android e iOS.
